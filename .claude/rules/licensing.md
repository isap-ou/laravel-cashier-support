# Licensing — No Copyleft Dependencies

This package is MIT-licensed. Copyleft licenses are incompatible.

- **NEVER** add dependencies with GPL, LGPL, AGPL, or any copyleft license
- **ALWAYS** verify the license of a new dependency before adding it to composer.json
- Allowed licenses: MIT, BSD-2-Clause, BSD-3-Clause, Apache-2.0, ISC, Unlicense
- If unsure about a dependency's license, check its `composer.json` or `LICENSE` file first
- Do NOT copy code from copyleft-licensed projects
- Do NOT use copyleft-licensed dev dependencies either — keep the entire dependency tree clean
